Running the tests

scale.factor <- 0.5

The package allows you to run benchmark tests on several different installations of R on the same machine.

This vignette scales the test sets down to 50% of standard size, to speed up the process for sake of illustration. For a full-scale test, adjust the tests by setting scale.factor = 1.
